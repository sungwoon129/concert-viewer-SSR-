## 공연 목록 뷰어

KOPIS(공연 예술 통합전산망)으로부터 API 요청을 통해 전국 각지의 다음 주(일~토) 공연일정 및 장소를 확인할 수 있는 웹 애플리케이션입니다.
OPEN API 제약으로 일 최대 1500건의 데이터만 호출가능합니다.

## 개발 환경

- SERVER - Next.js (SSR)
- IDE - Visual Studio Code
